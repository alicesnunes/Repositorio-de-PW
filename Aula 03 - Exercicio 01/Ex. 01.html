    Linguagens utilizadas

  • 1. HTML

    • O HTML não é uma linguagem de programação, mas sim uma linguagem de marcação de texto, que se produz através das TAGs. Utiliza-se essa marcação para produzir as páginas dos sites e os documentos eletrôrnicos que enviamos pelo email por exemplo. Entretanto, o HTML NÃO É A ÚNICA LINGUAGEM DE MARCAÇÃO que existe.

  • 2. CSS

    • A CSS está relacionada ao estilo da página, é como se a mesma estivesse dentro do desenvovimento de marcação. A CSS é basicamente o que deixa a página mais bonita e apresentável, dando cores à ela e deixando as TAGs organizadas estruturalmente.
